# MyStrokeApp 📱
MyStrokeApp เป็นแอพพลิเคชั่นที่พัฒนาขึ้นเพื่อช่วยในการฟื้นฟูสมรรถภาพของผู้ป่วยโรคหลอดเลือดสมอง โดยมีการใช้งานผ่านทางแอพพลิเคชั่นบนมือถือ โดยผู้ใช้สามารถทำการบำบัดด้วยการเล่นเกมในแอพพลิเคชั่น โดยแอพพลิเคชั่นจะทำการบันทึกผลการเล่นของผู้ใช้ และทำการแสดงผลให้ผู้ใช้เห็นผลการเล่นเกมของตนเอง และสามารถแสดงผลการเล่นเกมของผู้ใช้ในรูปแบบกราฟได้
